## II. HTML Skeleton
You might need this to follow along with any demos - below is the bare minimum markup needed for a valid HTML document:

```html
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="utf-8" />
	<title></title>
</head>
<body>

</body>
</html>
```
